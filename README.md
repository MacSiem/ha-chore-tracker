# HA Chore Tracker

![HA Chore Tracker Screenshot](screenshot.png)

Track household chores and tasks with assignment, scheduling, and completion tracking.

## Installation

### HACS (recommended)

1. Open HACS in Home Assistant
2. Go to **Frontend** section
3. Click the three dots menu > **Custom repositories**
4. Add `https://github.com/MacSiem/ha-chore-tracker` as **Dashboard**
5. Install **HA Chore Tracker**
6. Restart Home Assistant

### Manual

Copy the contents to `/config/www/community/ha-chore-tracker/`
